# VoteSmart

Interactive Election Platform built with React, Vite, and TypeScript.

## Live Demo
https://votesmart-898928325653.us-central1.run.app

## Features
- AI Chat Assistant
- Interactive Timeline
- Scenario Simulator
- Gamified Quizzes
- Flashcards

## Local Development
Run `npm install` and `npm run dev` to start the local development server.
